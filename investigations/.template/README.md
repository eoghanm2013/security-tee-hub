# Investigation Template

When starting a new investigation, copy this folder structure:

```
investigations/
└── SCRS-XXXX/
    ├── notes.md        # Your investigation notes, findings, questions
    ├── summary.md      # Final summary (populated on archive)
    └── assets/         # Supporting files
        ├── logs/       # Log files, agent logs
        ├── flares/     # Agent flares
        └── screenshots/# Screenshots, recordings
```

## Quick Start

To create a new investigation folder:
```bash
cp -r investigations/.template investigations/SCRS-XXXX
```

Or just ask Cursor: "Start investigation for SCRS-XXXX"
